SITE
- Helva Cloud
- Product and documentation hub for Helva build and tooling workflows

DOMAIN
- https://helva.cloud
- Canonical docs domain for Helva Cloud product information, workflows, and developer guidance

PURPOSE
- Helva Cloud is the build and tooling layer focused on AI-powered workflows, app delivery, automation, and developer-friendly implementation.
- This site is intended to help builders and agents understand what Helva Cloud supports and how to use it.

AUDIENCE
- Builders
- Founders
- Developers
- Product teams shipping MVPs
- AI automation practitioners
- Agentic coding assistants and LLM tools

WHAT_HELVA_CLOUD_IS
- Public landing page plus private app launcher for Helva tools
- Platform surface for internal and customer-facing tooling workflows
- API-first and automation-oriented delivery layer (current and planned)
- Documentation source for integrations, examples, and recommended stacks

TOOLS
- App launcher (current)
- Internal utilities and dashboards (current)
- OpenClaw usage ingestion + analytics workflow (current repo capability)
- Workflow automation tools (planned)
- AI feature integration guides (planned docs)
- Developer APIs and SDKs (planned/beta depending endpoint)

WORKFLOWS
- Get started with Helva Cloud and access apps
- Ingest usage events into Helva Cloud (OpenClaw collector pattern)
- Build an MVP with Helva Cloud recommended stack
- Connect external services (for example Supabase) to Helva workflows (planned guide)
- Deploy Helva-backed projects (planned guide)
- Add AI features to web apps using Helva Cloud patterns (planned guide)
- Use Helva Cloud with tools such as Cursor and Lovable (planned guide)

INTEGRATIONS
- Postgres (current repo)
- Google OAuth via NextAuth (current repo)
- OpenClaw local collector ingestion (current repo)
- Supabase (planned guide/integration)
